# Themisto

A simple crawling application using Koa 2, ES6.



## Getting Started

```
git clone https://github.com/theblackdude/themisto.git
cd themisto
mv .env.sample .env
npm run dev // or yarn dev
```

By default the API server starts on port 3000, http://localhost:3000.

### Prerequisites

* node >= v7.6.0
